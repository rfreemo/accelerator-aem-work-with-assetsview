# APL Theme

This directory contains the theme files for APL (Adobe Premium Learning) guides.

## Contents

- `guide-theme.css` - Main stylesheet (generated by build process)
- `fonts/` - Adobe Clean font files
- `images/` - Theme images and logos
- `templates/` - Document templates
- `frontmatter-*.md` - Frontmatter templates for different guide types

## Build Process

This directory is populated by the CSS build process. Run `npm run build:css` to generate the theme files.
